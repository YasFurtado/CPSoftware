Centro de Pesquisa em Software

O Centro de Pesquisa em Software (CPSoftware) é um grupo de pesquisa e desenvolvimento de software do Instituto Federal de Alagoas (IFAL), o qual se concentra no desenvolvimento de produtos de software e pesquisas nas áreas de engenharia de software, inteligência artificial e linguagens de programação.


Desde 2017

Pesquisa

Pesquisas nas áreas de engenharia de software, inteligência artificial, garantia de qualidade, Internet das coisas, and big data.

Soluções

Desenvolvimento de sistemas de software em várias linguagens de programação, protótipos em hardware e automação.

Análise de Dados

Análise de grandes quantidades de dados para encontrar informações úteis que possam ser utilizadas para tomada de decisão.
